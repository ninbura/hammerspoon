# setup
#### ⚠️ Warning, this will delete your current Hammerspoon configuration if you have one. ⚠️

1. ```zsh
   rm -rf ~/.hammerspoon
   ```
2. ```zsh
   git clone https://github.com/ninbura/hammerspoon ~/.hammerspoon
   ```
3. ```zsh
   brew install hammerspoon
   ```
